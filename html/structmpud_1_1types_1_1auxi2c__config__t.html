uint8_t mpud::types::auxi2c_config_t::sample_delay

Number of samples to delay on Aux i2c trasactions, max = 31, formula: rate = (sample_rate / delay + 1).

(e.g. sample_delay = 4, aux i2c transaction will occour after 4 samples. If sample rate is 1KHz, the Aux i2c transaction rate will be 1000 / (4 + 1) = 200 Hz) set zero if no delay is needed, so the transaction rate will be the same as sample rate.
